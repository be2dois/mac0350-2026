BetNotas: sistema de apostas em notas de suas avaliações com uma pseudomoeda própria a forte tendencia a percas para ensinar como apostar é ruim. 

Funcionalidades:
  - cadastrar materias
  - cadastrar uma avaliação e fazer uma aposta na sua nota
  - cadastrar uma nota e resolver a aposta
  - ver a lista de materias
  - ver lista de apostas de cada materia
  - saldo gerenciado automaticamente

Uso do mini projeto da Aula 9 de base.
