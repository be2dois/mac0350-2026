BetNotas: sistema de apostas em notas de suas avaliações com uma pseudomoeda própria, acompanhado de mensagens de desincentivo a apostas. 

Funcionalidades:
  - cadastrar uma avaliação e fazer uma aposta
  - cadastrar uma nota e resolver a aposta
  - ver lista de apostas em aberto e apostas passadas
  - saldo gerenciado automaticamente
